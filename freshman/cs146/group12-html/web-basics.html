Web Basics

Networking

Network Definition:

  • A network can be defined as two or more computers connected together in such a way that they can share resources. twofold
  • A network can be defined as two or more computers connected together in such a way that they can share resources.
  • Physical, through wires, cables, and wireless media
  • Logical, through the transport of data across the physical media

Resources

Purpose of a network is to share resources, aka: files, folders, printers, drives, anything else on a computer

Why build a network?

  • faster + less expensive communication
  • many users - requirements of a standardized exchange format
  • backup & recovery support: copies of data

disadvantages

  • Network Hardware, Software and Setup Costs
  • Hardware and Software Management and Administration Costs
  • undesirable sharing / behavior
  • data security concerns

Types of networks:

  • Local Area Network (LAN) - small geographic area, e.g. home, office, group of buildings
  • Wide Area Network (WAN) - broad area geographic, e.g. metropolitan, regional, countries
  • Metropolitan Area Network (MAN) - Geographic based network -- e.g. a single city

Intranet: private network contained within an enter

Internet: worldwide system of networks, network of networks in which users at any one computer can, if they have permission, get information from any other computer

HTML

History

HTML stands for HyperText Markup Language. HTML was created on the foundation of tags used in previous languages such as eXtensible Markup Language (XML). The language created a standardized system for content on the World Wide Web.

HTML was created in 1991 however it has been updating periodically, with the most recent update in 2016 providing HTML 5.1.

Tags

Some common tags that are the foundation to HTML include:

<DOCTYPE html>

This defines the file to be HTML5.

<html>

This is the root element of the HTML page.

<head>

This contains meta information about the document.

<title>

This defines a title for the document.

<body>

This tag is followed by visible page content
